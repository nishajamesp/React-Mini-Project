# RecipeHub 🍳

A modern and responsive **Recipe Sharing Platform** built with **React**, **Redux Toolkit**, **React Router**, and **Tailwind CSS**.  
RecipeHub allows users to browse delicious recipes, search by recipe name or ingredients, view recipe details, and save favorite recipes in a clean and user-friendly interface.

## 📖 Project Description

**RecipeHub** is a frontend React mini project developed as part of a practical assignment to demonstrate:

- React component-based architecture
- Global state management using **Redux Toolkit**
- Client-side routing with **React Router DOM**
- Modern UI styling using **Tailwind CSS**
- Responsive layout for mobile, tablet, and desktop

The application focuses on creating a visually appealing and professional recipe browsing experience where users can:

- Explore recipe cards
- Search recipes quickly
- Open recipe detail pages
- Save favorite recipes based on user
- Enjoy a clean and responsive UI

---

## ✨ Features

### Core Features
- 🍽️ Browse recipe cards with images and ratings
- 🔍 Search recipes by recipe name also can search recipie by talking
- 📄 View detailed recipe page for each recipe
- ❤️ Add recipes to favorites using Redux Toolkit
- 🧭 Smooth navigation using React Router DOM
- 📱 Fully responsive design for all screen sizes

### UI / UX Features
- 🎨 Modern, professional card-based layout
- 🖼️ High-quality recipe image presentation
- 🔥 Hover effects and clean spacing
- 🧡 Warm food-themed color palette
- ⚡ Fast and lightweight frontend experience with Vite
